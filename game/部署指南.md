# 暴风雪山庄游戏系统 - 详细部署指南（小白版）

## 📋 准备工作

### 你需要准备的设备
1. **1台电脑**（Windows/Mac/Linux都可以）- 作为主持人端和服务器
2. **10台Pad**（iPad或安卓平板）- 作为玩家端
3. **同一个WiFi网络** - 所有设备必须连接同一个WiFi

### 你需要安装的软件
1. **Node.js**（运行服务器用）
2. **代码编辑器**（推荐VS Code，免费且好用）

---

## 第一步：安装Node.js

### Windows用户
1. 打开浏览器，访问 https://nodejs.org/
2. 点击大大的绿色按钮 "Download Node.js (LTS)"
3. 下载完成后，双击安装包
4. 一路点击 "Next" → "Next" → "Install"
5. 安装完成后，按 `Win + R`，输入 `cmd`，回车
6. 在黑色窗口中输入：`node -v`
7. 如果显示版本号（如 v18.x.x），说明安装成功！

### Mac用户
1. 打开浏览器，访问 https://nodejs.org/
2. 点击 "Download Node.js (LTS)"
3. 下载完成后，双击.pkg文件
4. 一路点击 "继续" → "安装"
5. 打开"终端"（按 `Command + 空格`，输入 `terminal`）
6. 输入：`node -v`
7. 如果显示版本号，说明安装成功！

## 第三步：启动服务器

### Windows用户
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入以下命令（注意替换成你的实际路径）：
```bash
cd C:\Users\你的用户名\Desktop\snowstorm-game\server
```
3. 然后输入：
```bash
npm install
```
4. 等待安装完成（可能需要几分钟）
5. 安装完成后，输入：
```bash
node websocket-server.js
```
6. 如果看到 `WebSocket服务器已启动，端口: 8080`，说明成功！

### Mac用户
1. 打开"终端"
2. 输入以下命令（注意替换成你的实际路径）：
```bash
cd ~/Desktop/snowstorm-game/server
```
3. 然后输入：
```bash
npm install
```
4. 等待安装完成
5. 安装完成后，输入：
```bash
node websocket-server.js
```
6. 如果看到启动成功的提示，说明成功！

**⚠️ 重要：这个黑色窗口不能关闭！关闭后服务器就停止了！**

---

## 第四步：获取电脑IP地址

### Windows用户
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入：`ipconfig`
3. 找到 "IPv4 地址"，记下这个地址（如 192.168.1.100）

### Mac用户
1. 打开"系统偏好设置" → "网络"
2. 点击你连接的WiFi
3. 记下IP地址（如 192.168.1.100）

**把这个IP地址记下来，等下要用！**

---

## 第五步：修改配置

1. 用VS Code打开 `snowstorm-game` 文件夹
2. 找到 `src/store/websocketStore.ts` 文件
3. 找到第22行：
```typescript
const SERVER_URL = 'ws://localhost:8080';
```
4. 改成你的IP地址：
```typescript
const SERVER_URL = 'ws://192.168.1.100:8080';  // 改成你刚才记下的IP
```
5. 保存文件（按 `Ctrl + S` 或 `Command + S`）

---

## 第六步：启动前端

### 打开新的命令行窗口
**注意：不要关闭之前的服务器窗口！**

## 第七步：开始游戏

### 主持人端（电脑）
1. 打开浏览器（推荐Chrome）
2. 访问 `http://localhost:5173`
3. 选择"我是主持人"
4. 记下房间码（如 ABCD）

### 玩家端（10台Pad）
1. 确保所有Pad连接同一个WiFi
2. 打开浏览器（Safari或Chrome）
3. 访问 `http://192.168.1.100:5173`（用你的IP替换）
4. 选择"玩家加入"
5. 输入房间码
6. 输入玩家名字

### 游戏流程
1. 等10个玩家都加入后，主持人点击"开始游戏"
2. 主持人分配身份（点击玩家卡片选择身份）
3. 分配完成后，点击"开始游戏"
4. 按照游戏流程进行：自由阶段 → 行动阶段 → 结算阶段
5. 共5轮，结束后显示结果

## 常见问题

### Q1: 服务器启动失败，提示端口被占用
**解决方法**：
1. 打开 `server/websocket-server.js`
2. 找到第15行：`const PORT = 8080;`
3. 改成其他端口，如 `const PORT = 3000;`
4. 同时修改前端配置中的端口

### Q2: Pad无法连接到电脑
**解决方法**：
1. 确保所有设备在同一个WiFi
2. 检查Windows防火墙是否关闭
3. 尝试用电脑浏览器访问 `http://192.168.1.100:5173` 看能否打开

### Q3: npm install 很慢或失败
**解决方法**：
1. 使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
```
2. 然后再运行 `npm install`

### Q4: 如何停止服务器
**解决方法**：
1. 在命令行窗口按 `Ctrl + C`
2. 输入 `y` 确认
