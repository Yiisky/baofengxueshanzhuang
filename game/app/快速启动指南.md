# 暴风雪山庄 - 快速启动指南

## 最简单的方式：双击启动

### 步骤1：双击启动文件
找到文件夹中的 `启动游戏.bat` 文件，**双击运行**。

### 步骤2：等待自动配置
程序会自动：
- 安装必要的依赖
- 启动WebSocket服务器
- 启动前端服务器
- 打开浏览器

### 步骤3：开始游戏
1. 在打开的网页中点击 **"我是主持人"**
2. 系统会自动生成房间码（显示在页面右上角）
3. 告诉玩家房间码，让他们加入

---

## 玩家如何加入

### 方式一：同WiFi下访问（推荐）
1. 确保所有Pad和主持人电脑连接**同一个WiFi**
2. 在Pad浏览器中输入主持人电脑的IP地址
   - 格式：`http://[主持人IP]:5173`
   - 例如：`http://192.168.1.100:5173`
3. 输入房间码加入

### 方式二：查看启动窗口中的地址
启动批处理文件会显示主持人电脑的IP地址，直接告诉玩家即可。

## 常见问题

### Q: 双击后黑窗口一闪就消失了？
**A:** 可能是以下原因：
1. **没有安装Node.js** - 请先安装Node.js（https://nodejs.org/）
2. **文件路径包含中文或特殊字符** - 将文件夹移动到纯英文路径

### Q: 玩家无法连接？
**A:** 检查以下几点：
1. 所有设备必须在**同一个WiFi网络**
2. 关闭Windows防火墙或添加例外
3. 检查路由器是否开启了AP隔离

### Q: 如何查看本机IP？
**A:** 启动批处理文件会自动显示IP地址。也可以：
1. 按 `Win + R`
2. 输入 `cmd` 回车
3. 输入 `ipconfig` 回车
4. 查看 "IPv4地址"

### Q: 如何停止服务？
**A:** 
1. 在启动窗口中按任意键
2. 或直接关闭所有命令行窗口
3. 或打开任务管理器结束node.exe进程

## 手动启动（高级用户）

如果自动启动失败，可以手动启动：

### 1. 启动WebSocket服务器
```bash
cd server
npm install
node websocket-server.js
```

### 2. 启动前端（新窗口）
```bash
npm install
npm run dev
```

### 3. 打开浏览器
访问 `http://localhost:5173`

---

## 技术支持

如有问题，请检查：
1. Node.js版本 >= 18.0
2. 端口5173和8080未被占用
3. 防火墙设置
