# 暴风雪山庄游戏 - 素材替换指南

## 📁 素材存放位置

所有素材都放在 `public/images/` 文件夹下：

```
public/images/
├── main-bg.jpg          # 主背景图
├── role-icons.png       # 角色图标集合
├── map-overview.jpg     # 地图总览
└── ui-elements.png      # UI按钮和图标
```

---

## 🖼️ 如何替换素材

### 方法一：直接替换文件（最简单）

1. 找到 `public/images/` 文件夹
2. 把你想要替换的图片重命名为对应的名字
3. 直接覆盖原文件
4. 刷新浏览器即可看到效果

**注意**：图片格式要保持一致（.jpg 或 .png）

### 方法二：修改代码引用（更灵活）

如果你想用不同的文件名，需要修改代码：

**示例**：把 `main-bg.jpg` 换成你自己的背景

1. 把你的图片放到 `public/images/` 文件夹，比如叫 `my-bg.png`
2. 打开 `src/App.css` 或对应的组件文件
3. 找到背景引用的地方：
```css
background-image: url('/images/main-bg.jpg');
```
4. 改成：
```css
background-image: url('/images/my-bg.png');
```

```
直接替换 public/images/main-bg.jpg
```

### 2. 角色图标 (role-icons.png)

**用途**：显示各角色头像

**建议尺寸**：1000 x 1000 像素

**包含角色**（从左到右，从上到下）：
1. 凶手（红色面具）
2. 帮凶（黑色兜帽）
3. 侦探（放大镜）
4. 工程师（齿轮）
5. 黑客（电脑）
6. 医生（红十字）
7. 推理迷（问号大脑）
8-10. 备用位置

**替换方法**：
```
直接替换 public/images/role-icons.png
```

**如果你想单独替换某个角色图标**：

1. 准备10个单独的图标文件，比如：
   - `killer.png`
   - `accomplice.png`
   - `detective.png`
   - ...

2. 打开 `src/data/roles.ts`

3. 修改 `getRoleIcon` 函数：
```typescript
export function getRoleIcon(role: Role): string {
  const icons: Record<Role, string> = {
    killer: '/images/killer.png',
    accomplice: '/images/accomplice.png',
    detective: '/images/detective.png',
    engineer: '/images/engineer.png',
    hacker: '/images/hacker.png',
    doctor: '/images/doctor.png',
    fan: '/images/fan.png',
    good_fan: '/images/good_fan.png',
    bad_fan: '/images/bad_fan.png'
  };
  return icons[role] || '/images/default.png';
}
```

## ❓ 常见问题

### Q1: 替换后图片不显示？
**解决**：
1. 检查文件名是否正确
2. 检查文件是否在 `public/images/` 文件夹
3. 刷新浏览器（按 `Ctrl + F5` 或 `Command + Shift + R`）

### Q2: 图片变形了？
**解决**：
1. 检查图片尺寸是否符合推荐尺寸
2. 修改CSS中的 `background-size` 属性

### Q3: 图片加载很慢？
**解决**：
1. 压缩图片大小（推荐用 tinypng.com）
2. 使用 JPG 格式代替 PNG（如果不需要透明）
